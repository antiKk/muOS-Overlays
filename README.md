# muOS Overlays
A collection of Overlays for use in muOS on the Anbernic RG35XX series handhelds.

All Overlays have a pixel grid variant.
> [!NOTE]
> Pixel grid pattern sourced from https://github.com/Jeltr0n/Retro-Overlays
